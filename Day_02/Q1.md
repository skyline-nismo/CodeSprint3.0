## 📖 Problem Statement

Nitin is a huge cricket fan and never misses a match. During the IPL season, his professor assigns him **N** batting scores from different matches. Nitin notices that some players scored more than the team's average. He wants to find exactly how many players beat the team average so he can argue with his friends about who the real MVP is.

Your task is to help Nitin determine how many players scored **strictly above** the average of all **N** scores.

---

## 📥 Input Format

* The first line contains an integer **N** — the number of players.
* The second line contains **N** space-separated integers representing each player's score.

---

## 📤 Output Format

* Print a single integer — the count of players who scored **strictly above** the average.

---

## 💡 Example

### Input

```text
N = 5
Scores = 40 80 60 90 50
```

### Output

```text
2
```

---

## 📌 Constraints

| Constraint        | Value                 |
| ----------------- | --------------------- |
| Number of players | `1 ≤ N ≤ 10^5`        |
| Player score      | `0 ≤ scores[i] ≤ 100` |

---

## 💭 Hints

<details>
<summary><strong>Hint 1</strong></summary>

Compute the average in a first pass, then count in a second pass how many scores are strictly greater than it. Watch out—the average may be a decimal!

</details>
