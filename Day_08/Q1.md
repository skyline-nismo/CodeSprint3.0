## 📖 Problem Statement

Rahul’s professor has a very strange attendance rule during coding lectures. Every student’s roll number is checked in binary form before class begins.

The professor says:

* If the binary representation of a roll number contains an **odd number of ON bits (1s)**, the student must answer questions in class today.
* Otherwise, the student is considered **SAFE**.

Rahul has been asked to quickly identify which students are:

* **ANSWER**
* **SAFE**

Help him automate the process before the professor starts calling names.

---

## 📥 Input Format

* The first line contains an integer **N**.
* The second line contains **N** space-separated integers representing the students' roll numbers.

---

## 📤 Output Format

* For every roll number, print:

  * `ANSWER`, or
  * `SAFE`
* Print each result on a new line.

---

## 💡 Example

### Input

```text
4
3 5 8 7
```

### Output

```text
SAFE
SAFE
ANSWER
ANSWER
```

---

## 📌 Constraints

| Constraint         | Value               |
| ------------------ | ------------------- |
| Number of students | `1 ≤ N ≤ 10^5`      |
| Roll number        | `0 ≤ arr[i] ≤ 10^9` |

---

## 💭 Hints

<details>
<summary><strong>Hint 1</strong></summary>

Convert each number into its binary representation and count how many `1`s (ON bits) it contains.

</details>

<details>
<summary><strong>Hint 2</strong></summary>

If the count of ON bits is **odd**, print `ANSWER`.

</details>

<details>
<summary><strong>Hint 3</strong></summary>

If the count of ON bits is **even**, print `SAFE`.

</details>
