## 📖 Problem Statement

Souvik is building a secure digital firewall for CodeSprint 3.0.

To test the firewall strength, he designed a special numeric pattern.

The firewall follows these rules:

* Numbers increase from `1` on the left side.
* Numbers decrease on the right side.
* The middle section is filled with `@` symbols.

As the rows increase:

* The number section grows.
* The `@` section shrinks.

Help Souvik generate the firewall pattern.

---

## 📥 Input Format

* A single integer `N`.

---

## 📤 Output Format

* Print the required firewall pattern.

---

## 💡 Example

### Input

```text
N = 5
```

### Output

```text
1@@@@@@@@1
12@@@@@@21
123@@@@321
1234@@4321
1234554321
```

---

## 📌 Constraints

| Constraint   | Value        |
| ------------ | ------------ |
| Value of `N` | `1 ≤ N ≤ 20` |

---

## 💭 Hints

<details>
<summary><strong>Hint 1</strong></summary>

Every row has three parts: left numbers, middle `@` symbols, and right numbers.

</details>

<details>
<summary><strong>Hint 2</strong></summary>

In each new row, the number of `@` symbols decreases by `2`.

</details>
