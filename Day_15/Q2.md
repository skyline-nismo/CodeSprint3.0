## 📖 Problem Statement

A coastal rescue team lead, Tannu, is evacuating people from an island before a storm arrives.

Each rescue boat can carry exactly two people, and for safety reasons, the total weight of the two people assigned to the same boat determines the load on that boat.

To keep the evacuation as safe as possible, the team wants to arrange the people so that the heaviest-loaded boat is as light as possible.

You are given the weights of **N** people (where **N** is even). Help Tannu determine the minimum possible value of the maximum boat load if everyone must be assigned to a boat and each boat carries exactly two people.

---

## 📥 Input Format

* The first line contains an integer **N** — the number of people.
* The second line contains **N** space-separated integers representing the weights of the people.

---

## 📤 Output Format

* Print a single integer — the minimum possible value of the maximum boat load.

---

## 💡 Example

### Input

```text
6
70 50 80 60 40 90
```

### Output

```text
130
```

---

## 📌 Constraints

| Constraint                 | Value               |
| -------------------------- | ------------------- |
| Number of people (`N`)     | `2 ≤ N ≤ 2 × 10^5`  |
| Number of people           | `N is even`         |
| Weight of the *i*th person | `1 ≤ Weight ≤ 10^9` |

---

## 💭 Hints

<details>
<summary><strong>Hint 1</strong></summary>

If two heavy people are placed together, one boat may become unnecessarily overloaded.

</details>

<details>
<summary><strong>Hint 2</strong></summary>

Sort the weights first.

</details>

<details>
<summary><strong>Hint 3</strong></summary>

Pair the lightest remaining person with the heaviest remaining person.

</details>

<details>
<summary><strong>Hint 4</strong></summary>

Use two pointers: one at the beginning of the sorted array and one at the end. Move both pointers inward after forming each pair.

</details>
