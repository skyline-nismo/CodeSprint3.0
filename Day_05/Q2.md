## 📖 Problem Statement

Harsh is helping his professor track attendance during a workshop in college.

Students enter the seminar hall batch by batch.

You are given an array where:

* `arr[i]` represents the number of students entering during batch `i`.

The seminar hall becomes full once the total number of students inside becomes **strictly greater than** `X`.

Your task is to find the **first batch index** where the hall becomes full.

If the hall never becomes full, print `-1`.

---

## 📥 Input Format

* First line contains two integers `N` and `X`.
* Second line contains `N` space-separated integers.

---

## 📤 Output Format

* Print a single integer:

  * Index of the first batch where the total number of students exceeds `X`.
  * Print `-1` if it never happens.

---

## 💡 Example

### Input

```text
N = 6
X = 20
3 4 5 2 8 1
```

### Output

```text
4
```

---

## 📌 Constraints

| Constraint              | Value               |
| ----------------------- | ------------------- |
| Number of batches       | `1 ≤ N ≤ 10^5`      |
| Hall capacity threshold | `1 ≤ X ≤ 10^9`      |
| Students per batch      | `0 ≤ arr[i] ≤ 10^4` |

---

## 💭 Hints

<details>
<summary><strong>Hint 1</strong></summary>

Think of it like a live attendance counter increasing batch by batch.

</details>

<details>
<summary><strong>Hint 2</strong></summary>

You don’t need to store totals for every batch separately. One running total is enough!

</details>
