## 📖 Problem Statement

After the final exams were over, **Ritrisha** received her marks for all subjects.

While checking her report card, she became curious and wondered:

> **Did I score the same marks in any two subjects?**

Your task is to determine whether there are any duplicate marks in her report card.

---

## 📥 Input Format

- The first line contains an integer **`N`** — the number of subjects.
- The second line contains **`N`** space-separated integers representing the marks obtained in each subject.

---

## 📤 Output Format

- Print **`YES`** if any duplicate marks exist.
- Otherwise, print **`NO`**.

---

## 💡 Example

### Input

```text
N = 6
Marks = 78 85 91 78 88 95
```

### Output

```text
YES
```

---

## 📌 Constraints

| Constraint | Value |
|------------|-------|
| Number of subjects | `1 ≤ N ≤ 10^5` |
| Marks | `0 ≤ marks ≤ 100` |

---

## 💭 Hint

<details>
<summary><strong>Hint 1</strong></summary>

Use a **frequency array** or a **hash set** to keep track of the marks that have already appeared.

If you encounter a mark that has already been seen, you can immediately print **`YES`** and terminate.

</details>
