## 📖 Problem Statement

During an adventure in an ancient kingdom, **Jagrit** discovered a magical pathway filled with glowing energy stones. Each stone had a number written on it.

Unfortunately, some stones had lost their magical power and displayed the number **`0`**.

The kingdom's wizard gave Jagrit a task:

> **Move all the powerless stones (`0`s) to the end of the pathway while keeping the order of all active stones exactly the same.**

Help Jagrit rearrange the stones correctly.

---

## 📥 Input Format

- The first line contains an integer **`N`** — the number of stones.
- The second line contains **`N`** space-separated integers representing the stones.

---

## 📤 Output Format

Print the updated arrangement after moving all zeroes to the end while preserving the relative order of the non-zero elements.

---

## 💡 Example

### Input

```text
N = 7
Elements = 1 0 4 0 5 2 0
```

### Output

```text
1 4 5 2 0 0 0
```

---

## 📌 Constraints

| Constraint | Value |
|------------|-------|
| Number of stones | `1 ≤ N ≤ 10^5` |
| Stone value | `−10^9 ≤ elements ≤ 10^9` |

---

## 💭 Hints

<details>
<summary><strong>Hint 1</strong></summary>

Use the **two-pointer technique**.

One pointer keeps track of the position where the next non-zero element should be placed, while the other traverses the array.

</details>

<details>
<summary><strong>Hint 2</strong></summary>

Whenever a non-zero element is found, move it forward.

After all non-zero elements have been processed, fill the remaining positions with **`0`**.

</details>
